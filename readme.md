## Installation

To get started with Reusic Backend, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/imran9663/reusic-backend.git
    cd reusic-backend
    ```

2. Install dependencies:

    ```bash
    npm install
    ```
